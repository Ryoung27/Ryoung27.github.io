All Budget Beta

A budgeting app to improve the loan operations and construction process for banks. All Budget Beta is designed to cut down on the go between of banks and their construction customers when creating an app. This was done in React and used the Create, Read, Update, and Delete methods.

Run for It

Run for It is a running diary. I am an avid runner and have never liked using the majority of running apps due to their bloatware, so I made a simple, sleek, running app that ran on the Django Framework of Python to do for me what other apps could not.
